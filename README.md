# elchat

elchat adalah platform chat web modern yang menghadirkan pengalaman komunikasi real-time yang cepat, intuitif, dan efisien.

Fitur utama meliputi:
- Pesan teks real-time
- Berbagi foto, file, dan voice note
- Kamera langsung dari browser
- Status online pengguna
- Dukungan instalasi sebagai Progressive Web App (PWA)

Dengan arsitektur berbasis web modern, elchat dapat diakses lintas perangkat dan siap dikembangkan menjadi aplikasi komunikasi yang scalable.
